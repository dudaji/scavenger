# Scavenger 프로그램 계획

## 개요
- 사용자가 정해놓은 시간대에 정해진 토큰 limit 까지 우선순위가 높은 태스크부터 자동으로 진행하는 프로그램.

## 목적
- 특정 시간대에만 토큰 사용량이 몰리고 그 외의 시간대에는 오히려 남기때문에 잉여 주간 토큰 사용량을 알차게 사용하기위해

## 기능 목록
1. 태스크 목록 입력 - 우선순위 존재
2. 설정 - 프로그램 실행 시간대(사용자 비활성 시간), 일자별 프로그램 토큰 제한 등
3. 긴급 정지 기능 - 사용자 요청시 바로 중단
4. 결과물 전달 - 일자별로 정해진 시간에 그날 작업한 결과물을 메신저(이메일 우선, 다른 메신저는 추후 개발)를 통해 전송
5. 태스크 수행시 사용자의 입력이 없으므로 알아서 처리하거나 중단해야할듯.
6. ralph loop 처럼 토큰 사용량이 많아지더라도 더 똑똑하게 작업을 진행할 수 있어야함.

## 사용 기술
- Python 기반
- 설정파일 저장 필요시 DB없이 로컬 저장소 사용(~/.scavenger 등)
- 구현을 단순화하기 위해 claude code만 일단 적용 대상. 추후 확장 가능함을 고려
- 우선 CLI만.

## 검증
- 진행할 태스크를 넣어놓고 정해진 시간대가 되었을 때 제대로 작동하는지 확인

## 추후 개발
- 패키지 형식으로 메모장 앱처럼 만들고 싶음. 즉, 메모장에 태스크 우선순위에 맞게 넣으면 비활성시간대에 알아서 작동하고 아침에 결과를 받을 수 있게.
